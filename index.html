🧬 TimeFlies v1.0 Documentation

Machine Learning for Aging Analysis in Drosophila Single-Cell Data
Complete framework with deep learning, SHAP analysis, and batch correction

🚀 Quick Start

# Install TimeFlies curl -O https://raw.githubusercontent.com/rsinghlab/TimeFlies/main/install_timeflies.sh chmod +x install_timeflies.sh ./install_timeflies.sh # Activate and run source .activate.sh timeflies setup timeflies train

🔬 Research Workflow

  1. Data Setup: Add your *_original.h5ad files to data/project/tissue/
  2. Configuration: Edit configs/default.yaml
  3. Setup: Run timeflies setup for complete preparation
  4. Training: Run timeflies train with auto-evaluation
  5. Evaluation: Run timeflies evaluate to assess model performance
  6. Analysis: Results in outputs/ with SHAP interpretability
